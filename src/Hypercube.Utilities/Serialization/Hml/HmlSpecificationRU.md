# Hypercube язык разметки (.hml формат)

## Обзор

## Конструкции

### Идентификатор
Это символьная последовательность, предназначенная для обозначения элемента объекта.

#### Ограничения семантики
1. Первый символ идентификатора не может быть цифрой.
2. Идентификаторы чувствительны к регистру.
3. Зарезервированные символы не допускаются в качестве идентификаторов.

### Значение
Любой тип данных. Может быть присвоено идентификатору. 

## Типы данных

### Примитивы

#### Число (Number)
Этот тип данных представляет целые и дробные числа.

```yaml
1
1.5
0b1001
0xF5B6
5e-10
```

#### Строка (String)
Этот тип данных представляет строку или символ.

**1. Строка**
```yaml
"Hello"
```

**2. Символ**
```yaml
"A"
```

**3. Многострочная строка**
```yaml
"""
First line
Second line
"""
```

#### Булевое значение (Boolean)
Этот тип данных представляет булевое значение.

```yaml
true
false
```

#### Отсутствие значения (Null)
Этот тип данных представляет отсутствие значения.
Обычно употребляется либо с явным указанием типа, либо для сериализации шаблона.

```yaml
null
```

#### Перечисление Enum
Этот тип данных представляет перечисление чисел.
Поставляется в настройках сериализатора.

```yaml
$LogLevel.Info
```
---
### Коллекции
#### Объект (Object)
Этот тип данных представляет пару идентификатор-значение.
Идентификатор уникален в пределах одного объекта.

```yaml
{
  field1: "Hi";
  field2: "World";
}
```

#### Словарь (Dictionary)
Этот тип данных представляет собой коллекцию пар ключ–значение, где:
1. Ключ и значение может быть любым типом данных.
2. Каждый ключ в пределах одного словаря уникален.

```yaml
[
  "Hi": "World";
  1: 0;
  true: false;
  null: null;
  unknown: unknown;
]
```

#### Список (List)
Этот тип данных представляет собой коллекцию значений.

```yaml
[
  "World";
  0;
  false;
  null;
  unknown;
  ["key1"; "a"];
  ["key2"; "ab"];
]
```

---
### Специальные
#### Неизвестность (Unknown)
Этот тип данных означает что тип данных не определён.
Один из способов получить неизвестность, это незарезервированная последовательность символов.

```yaml
Unknown?
```

```yaml
???
```

```yaml
Hello World!
```

```yaml
?123!;
```
